Linguagens Usadas.

HTML

HTML é uma linguagem de marcação usada para criar e estruturar páginas da web. É composto por tags que definem a estrutura e o conteúdo de uma página. O HTML é responsável pelo layout, aparência e links de uma página da web e é suportado por todos os navegadores.

CSS

CSS é uma linguagem de estilo utilizada para definir a aparência e o layout dos elementos HTML em uma página da web. Ele permite controlar cores, fontes, margens, posicionamento e outros aspectos visuais para criar um design atraente e consistente.

JavaScript

JavaScript é uma linguagem de programação utilizada para criar interatividade em páginas da web, adicionando comportamento dinâmico aos elementos HTML e manipulando eventos. É amplamente utilizado tanto no lado do cliente (navegador) quanto no lado do servidor.
